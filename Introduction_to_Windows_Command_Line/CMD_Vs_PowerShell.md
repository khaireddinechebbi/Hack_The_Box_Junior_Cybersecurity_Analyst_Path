## Question 1:
What command string can we use to view the help documentation for the command Get-Location?
```powershell
Get-Help Get-Location
```

## Question 2:
What command can we use to show us our current location on the host system?
```powershell
Get-Location
```

## Question 3:
What hotkey can be used to clear our input line completely?
```powershell
Escape
```
