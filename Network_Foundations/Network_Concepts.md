## Question 1:
What layer of the OSI model is responsible for physical connections like Ethernet cables?
```bash
Physical Layer
```

## Question 2: 
Name the OSI layer that deals with logical addressing and routing.
```bash
Network Layer
```

## Question 3: 
Which protocol ensures reliable delivery of data and operates at the Transport Layer?
```bash
TCP
```

## Question 4: 
At what layer do switches operate within the OSI model?
```bash
Data Link Layer
```

## Question 5: 
What layer of the TCP/IP model corresponds to the OSI model’s Application, Presentation, and Session layers?
```bash
Application Layer
```

## Question 6: 
Which layer of the OSI model manages data encryption and data format conversion?
```bash
Presentation Layer
```

## Question 7: 
Name a protocol used for web browsing that operates at the Application Layer.
```bash
HTTP
```

## Question 8: 
Which OSI layer ensures the segments are transferred reliably and in sequence?
```bash
Transport Layer
```

## Question 9:
Which protocol provides fast, connectionless communication and operates at the Transport Layer?
```bash
UDP
```
